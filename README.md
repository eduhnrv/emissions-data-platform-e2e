# Environmental Data Platform - End-to-End Emissions Analytics

#### Autor : Eduardo S. Henríquez N.
#### Fecha : 15 / 01 / 26.

## Descripción General.

Plataforma de datos orientada a la ingesta, orquestación, modelado y análisis de emisiones a cuerpos de agua en Chile, utilizando datos públicos de la Subsecretaría del Medio Ambiente.
